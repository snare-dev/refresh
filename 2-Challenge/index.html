HTML & CSS Basic Summary

HTML Summary

HTML(HyperText Markup Language) is used to define our pages' content, structure and meaning. You dont use HTML for styling, we use CSS instead.

  • HTML uses "elements" to describe content.
  • HTML elements typically have an opening and a closing tag.
  • You can also have void(empty) tags like the image(img) tag.
  • You can also configure elements with attributes.
  • There is a long list of available elements but, with time you'll gain the experience needed.

You can also explore all available elements in the MDN HTML Elements Reference.

CSS Summary

CSS (Cascading Style Sheets) is used for styling your page content.

  • Styles are assigned property-value pairs.
  • You can assign styles via the "style" attribute.
  • To avoid code duplication you typically use global styles(e.g via the "style" element) instead.
  • Alternatively, you can work with external stylesheets which you then "link" to.
  • When working with CSS, concepts like "inheritance", "specificity" and "box model" should be well understood.

Learn more about all available CSS properties and values on the MDN CSS property reference.
